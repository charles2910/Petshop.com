ATENÇÃO: Ao abrir o site pela primeira vez é necesário esperar um tempo até que todos
os produtos sejam cadastrados no IndexedDB, uma vez que a implementação é local, assim que
os produtos forem carregados o site funcionará automaticamente.

Para navegar pelo site é necessário criar um servidor local

Descrição dos arquivos:
	Pastas 
		conteudos: todos os conteudos do que são carregados dinamicamente
		css: arquivos de estilo do site
		html: contém o index que é utilizado para navegar por todo site
		images: icones utilizados no site e imagens dos produtos
		js: scripts utilizados
		produtos: json com todos os produtos do site

	Scripts:
		Os arquivos AJAX.js e servidor.js contém diversas funções que mais tarde serão implementadas
		no servidor como: retornar os produtos requeridos, filtrar produtos, gravar dados, ler dados 
		e etc.
